##  Project Overview
A SaaS-based application designed to help job seekers optimize their resumes. Using **n8n** as the backend orchestrator, the tool analyzes Job Descriptions (JDs) and automatically tailors the user's resume to include relevant keywords and skills.

## ✨ Key Features
- **JD Matching:** AI-driven analysis to align resumes with specific job requirements.
- **Dynamic PDF Generation:** Converts optimized content into professional PDF layouts.
- **ATS Optimization:** Ensures resumes are formatted to pass through Applicant Tracking Systems.

## Tech Stack
- **Workflow Backend:** n8n.io
- **AI Engine:** OpenAI / Gemini
- **Document Processing:** Google Docs API / HTML-to-PDF converters
